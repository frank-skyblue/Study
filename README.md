## How To Add Notes

1. Add reusable techniques to `dsa/patterns/`.
2. Add solved coding problems to `dsa/problems/` using the `0001_problem_name.md` format.
3. Add system design building blocks to `system-design/fundamentals/`.
4. Add full system design walkthroughs to `system-design/case-studies/`.
5. Store diagrams in `assets/diagrams/` and link to them from the relevant note.
